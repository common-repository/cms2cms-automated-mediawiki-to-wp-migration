Plugin for automated data migration from MediaWiki to WordPress

== Description ==

Decided to migrate your website content from MediaWiki to WordPress? Entrust the process to CMS2CMS plugin and you’ll get your new WordPress with all your MediaWiki content.

= How does this plugin help: =
- Launching Free Demo Migration helps to preview the migration results first;
- Average migration time is from several minutes to several hours, depending on how many content items you’re up to move from MediaWiki to WordPress;
- Migrate the content you want. Map the data and choose how the migrated content lives and looks on WordPress;
- Complete migration price depends on the amount of content you’re going to move from MediaWiki to WordPress.

= WHAT WILL BE MOVED FROM MEDIAWIKI TO WORDPRESS? =
* Content along with content creation date and content author;
* Categories;
* Content images;
* Attachments (images and videos);
* Users, and their roles;
* Comments.

= PRESERVED SEO JUICE AFTER THE MIGRATION =
* Migration of SEO friendly URLs;
* Automated 301 redirects from existing MediaWiki website to the new WordPress website.

= COMPATIBLE WITH: WORDPRESS 4.1.X - 5.0.X. =
* Live help is available. Our support team is here 24/7.
* When a custom migration is needed, CMS2CMS offers data migration service packages to order.

== Installation ==

1. Download the plugin zip file.
2. Go to the Admin Panel -> Plugins -> Add New -> Upload Plugin.
3. Choose the downloaded file in zip format and click the “Install Now” button.
4. Go to Admin -> Plugins, activate the “CMS2CMS: Automated MediaWiki To WordPress Migration Plugin”.
5. Register an account or sign in to the existing one.
6. Provide the URL address of your Existing website.
7. Download the Connection Bridge and place it into the Existing website root folder. You can find a detailed instruction [here](https://cms2cms.com/faqs/what-is-the-connection-bridge-and-how-to-install-it/).
8. Go back to the plugin and click “Verify Connection”. You’ll be redirected to the CMS2CMS website to proceed with your migration.
